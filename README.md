# Приложение для концентрации внимания

Приложение для концентрации внимания с использованием техники Pomodoro. Разработано на Python с использованием библиотеки Flet.

## Функциональность

- Таймер для периодов концентрации
- Таймер для перерывов
- Визуальная анимация роста гриба во время работы
- Настраиваемое время работы и отдыха с помощью слайдера
- Сохранение настроек в базе данных
- Различные типы грибов для анимации (случайный выбор)

## Технологии

- **Python** - основной язык программирования
- **Flet** - создание GUI приложения
- **Pandas** - работа с базами данных
- **Random** - генерация случайных значений для анимации

## Структура проекта

- `main.py` - главный файл запуска приложения
- `menu_page.py` - страница главного меню
- `period_page.py` - страница рабочего периода
- `pause_page.py` - страница перерыва
- `transitional_pages.py` - страницы окончания периодов
- `read_dataframe.py` - работа с базами данных
- `timer.py` - логика таймеров
- `slider.py` - компонент слайдера для настройки времени
- `mushrooms.py` - анимация роста грибов

## Установка и запуск

1. Клонируйте репозиторий
```
git clone https://github.com/your-username/concentration-app.git
cd concentration-app
```

3. Установите зависимости
```
pip install -r requirements.txt
```

5. Запустите приложение
```
python main.py
```

## Использование

- На главном экране нажмите "Начать период"
- Во время работы вы будете видеть анимацию роста гриба
- После рабочего периода последует перерыв
- Цикл повторяется для повышения продуктивности
